# Hoofdstuk 1

Description

| Begrip | Omschrijving |
|------|--------------|
| Industrie | Een verzameling bedrijven die gelijkaardige activiteiten uitvoeren en soortgelijke producten of diensten aanbieden. |
| Core business | De kernactiviteit / reden van bestaan van een bedrijf; levert de belangrijkste toegevoegde waarde aan klanten of maatschappij. |
| KMO | Verzamelterm voor micro-, kleine en middelgrote ondernemingen. |
| Multinational | Zeer grote onderneming met vestigingen in meerdere landen. |
| Afdeling | Specifieke eenheid binnen een onderneming met eigen taken en verantwoordelijkheden. |
| Operationele afdelingen | Doen het bedrijf draaien: maken producten en leveren kern-diensten; meestal geen klantcontact. |
| Commerciële afdelingen | Doen het bedrijf groeien: brengen producten en diensten op de markt en verkopen ze; veel klantcontact. |
| Ondersteunende afdelingen | Faciliteren de werking van het bedrijf achter de schermen; geen contact met klanten. |
| Proces | Reeks activiteiten die in een vaste volgorde worden uitgevoerd om een specifiek resultaat te bereiken. |
| Automatisch proces | Proces dat volledig door systemen wordt uitgevoerd zonder menselijke tussenkomst. |
| Semiautomatisch proces | Proces waarbij mens en systeem samenwerken; deels automatisch, deels manueel. |
| Manueel proces | Proces waarbij alle stappen door mensen worden uitgevoerd. |
| CIO / CTO | IT-functies (Chief Information Officer / Chief Technology Officer) in grote organisaties. |
| ICT-manager / ICT-directeur | Leidinggevenden van de IT-afdeling in middelgrote en kleinere organisaties. |
| Software | Alle computerprogramma’s, van besturingssystemen tot applicaties en games. |
| Bedrijfsapplicatie | Software gericht op bedrijfsactiviteiten zoals klantenbeheer, facturatie, documentbeheer en opleidingen. |
| ERP | Enterprise Resource Planning: bundelt bedrijfsprocessen zoals voorraad, bestellingen en facturatie. |
| CRM | Customer Relationship Management: beheer van klanten, leads, offertes en marketingcampagnes. |
| DMS | Document Management System: beheer van documenten, versies en toegangsrechten. |
| CMS | Content Management System: beheer van websites en digitale content. |
| LMS | Learning Management System: beheer van opleidingen, cursussen, examens en opvolging. |
| Embedded software | Software die draait op toestellen anders dan pc’s of servers (bv. tv, digibox, spelconsole). |
| Configuratie | Functies aanpassen door bestaande instellingen te wijzigen. |
| Customisatie | Functies uitbreiden zonder code, bv. via modules of plugins. |
| Development | Uitbreiden of integreren met code, zoals koppelingen of maatwerk. |
| Cloud computing | Het leveren van computerdiensten (software, opslag, servers) via het internet in plaats van lokaal (on-premise). |
| Architectuur | Het algemene ontwerp van IT-, telecom- en netwerksystemen en hun samenhang binnen een organisatie. |
